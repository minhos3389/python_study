## 리스트

- ex) `리스트명 = [데이터, 데이터, 데이터]`
- 기차를 연상해봅니다.
- index는 `0`부터 시작합니다. (index를 이용해서 리스트의 몇 번째 데이터인지 알 수 있습니다.)

### 데이터 조작하기
- 데이터 추가 `리스트.append(데이터)` , 리스트의 마지막에 데이터가 추가됩니다.
- 데이터 할당 `리스트[인덱스] = 데이터` 
- 데이터 삭제 `del 리스트[인덱스]`

```Python
a = [1, 2, 3]
a.append(4)
print(a) # [1, 2, 3, 4]

a[0] = 0 
print(a) # [0, 2, 3, 4]

del a[0]
print(a) # [2, 3, 4]
```
- 슬라이싱 `리스트[시작: 끝+1]`
- 리스트 길이 `len(리스트)`
- 리스트 정렬 `리스트.sort()`

```Python
b = [3, 4, 2, 6]
b.sort()
print(b) # [2, 3, 4, 6]
```
