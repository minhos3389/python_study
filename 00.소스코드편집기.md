### 소스코드 편집기 사용하는 이유

- `폴더` 및 `파일`을 쉽게 정리가능
- `코드 자동완성` 기능
- `디버깅(오류수정)`이 쉽다.
- 유용한 `단축키`들이 많다.

### VS Code (Visual Studio Code) - Window

    웹 사이트에 visual studio code를 검색 후 알맞은 bit 운영체제에 맞는 version을 설치해줍니다.
    Openfolder를 클릭하고 폴더를 선택해주면 해당 폴더에서 vscode가 열린 상태가 됩니다.
    좌측 상단 EXPLORER에서 폴더 및 파일목록을 살펴볼 수 있고 버튼을 통해서 간단하게 폴더 및 파일을 생성할 수 있습니다.

    파일을 저장하기 전에는 열려있는 파일의 이름창 옆에 동그라미가 생기고, 저장시에는 사라집니다.

### VSCode 초기 상태로 변경
비주얼 스튜디오 코드를 쓰다보면 `초기화`된 상태로 돌리고 싶을 때가 있습니다.  
하지만 단순히 프로그램을 삭제하는 방법으로는 기존 설정이 그대로 남아 있어 문제가 됩니다.  
`C:\Users\user\.vscode` 폴더를 삭제해줍니다.  
`C:\Users\user\AppData\Roaming\Code` 폴더를 삭제해줍니다.  
그 다음 프로그램 삭제 후 재설치를 수행하면 초기화된 VSCode가 설치됩니다.
